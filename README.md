# YOLOv4 Fursuit Detector

Detects fursuits images: basically garbage-in garbage-out kinda thing...

Built using React, Flask, and PyTorch.

## Installation

Fork or clone it, then install it:

```sh
npm install

# or

npm i
```

Then, run it locally on your device with:

```sh
npm run start
```
