## Get it up and running locally

Clone this repo, cd into the `client` folder and install it via npm or yarn:

```sh
# cd from client
cd client

# Install that baby
npm i

# or install via yarn
yarn install
```

Then, start a local dev server by running:

```sh
npm run dev

# or install via yarn
yarn dev
```
